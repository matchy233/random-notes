# IDE tool tips

## JetBrains 系

### 特别常用的快捷键

| 快捷键 | 作用 |
| --- | --- |
| `Ctrl` + `Y` | 删除整行 |
| `Ctrl` + `Shift` + `Z` | 撤销 |

### 好用的插件

| 插件名 | 作用 | JB plugin marketplace 地址 |
| --- | --- | --- |
| Rainbow Brackets | 用彩色标注成对括号，再也不用担心大括号配不上对啦 | [10080-rainbow-brackets](https://plugins.jetbrains.com/plugin/10080-rainbow-brackets) |
